# 이롭게 바탕체 
이롭게에서 만든 이롭게 바탕체입니다. 이롭게 바탕체는 디지털 환경에 최적화 된 웹을 위한 폰트입니다. 한글 세리프 폰트로서는 처음으로 웹 전용으로 기획되고 제작되었습니다.
